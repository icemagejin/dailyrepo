# 背景风格对比 - 八字合盘海报

## 三种背景风格对比

### 1. 纯微纹理版（当前默认）

**特点**：
- 背景：#1A1A2E 深色 + 150个点 + 20条线
- 风格：现代极简
- 文化属性：通用美学

**优势**：
- ✅ 极简设计，信息聚焦
- ✅ 现代感强
- ✅ 不干扰内容
- ✅ 适合快速阅读

**适用场景**：
- 日常使用
- 现代审美偏好
- 信息优先

---

### 2. 中式传统版（刚才展示的）

**特点**：
- 背景：中式回纹边框 + 云纹四角 + 八卦图案 + 天干地支
- 风格：传统中式
- 文化属性：传统文化

**优势**：
- ✅ 文化气息浓厚
- ✅ 视觉层次丰富
- ✅ 装饰性强
- ✅ 符合八字主题

**适用场景**：
- 文化传播
- 传统节日
- 文化类应用

---

### 3. 纯色简洁版（另一种选择）

**特点**：
- 背景：纯色 #1A1A2E
- 风格：极简纯色
- 文化属性：现代极简

**优势**：
- ✅ 最简洁
- ✅ 文字最清晰
- ✅ 文件最小
- ✅ 加载最快

**适用场景**：
- 性能优先
- 极简主义
- 文字阅读

## 实现方式

### 可配置参数
```python
# 背景类型
BACKGROUND_TYPE = "minimal"  # minimal | traditional | solid

# 是否添加边框
ADD_BORDER = False  # True | False

# 是否添加四角装饰
ADD_CORNERS = False  # True | False

# 是否添加八卦图案
ADD_BAGUA = False  # True | False

# 是否添加天干地支
ADD_TIANGANDIZHI = False  # True | False
```

### 混合搭配
可以自由组合各种背景元素：
- 边框 + 四角装饰
- 边框 + 八卦图案
- 四角装饰 + 天干地支
- 边框 + 四角 + 八卦 + 天干地支（完整版）
