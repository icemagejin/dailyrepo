## 设计哲学

### 核心理念
- **诧寂（Wabi-Sabi）**：接受不完美、无常、不完整
- **极简**：去除一切不必要的装饰
- **留白**：大面积空白，营造孤独感和禅意
- **东方美学**：水墨意境，含蓄内敛

### 不完美之美
- 不追求对称、完美对齐
- 接受自然的随机性
- 纸质纹理、淡雅墨色
- 不具象的表达

## 1. 色彩系统

### 主色（墨色系）
```python
BACKGROUND = (250, 248, 245)  # 极淡暖白 (#FAF8F5)
TEXT_PRIMARY = (80, 75, 70)    # 中墨 (#504B46)
TEXT_SECONDARY = (130, 125, 118)  # 浅墨 (#827D76)
TEXT_TERTIARY = (170, 165, 158)   # 极淡墨 (#AAA59E)
```

### 色彩原则
- **单一配色**：只使用墨色系，不混用冷暖
- **饱和度 < 80%**：所有颜色保持低饱和度
- **克制使用**：不使用鲜艳颜色作为强调
- **自然质感**：纸张、墨色的自然过渡

### 禁止事项
- ❌ 纯黑色 `#000000`
- ❌ 紫色/蓝色 "AI 渐变"
- ❌ 过饱和颜色（> 80%）
- ❌ 多于一个强调色

## 2. 字体系统

### 字体家族
```python
PRIMARY_FONT = "Noto SansCJK"  # 主字体（支持中文）
FONT_REGULAR = "Noto SansCJK Regular"
FONT_MEDIUM = "Noto SansCJK Medium"
FONT_BOLD = "Noto SansCJK Bold"
```

### 字号体系
```python
# Display 层级（超大标题）
DISPLAY_SIZE = height * 0.22  # 主日期

# Body Large 层级（重要信息）
BODY_LARGE_SIZE = width * 0.045  # 农历
BODY_LARGE_SIZE_2 = width * 0.055  # 八字

# Body 层级（正文）
BODY_SIZE = width * 0.040  # 英文月周、警语、福语
```

### 字重层级
- **Display (SemiBold/Bold)**：主日期，视觉焦点
- **Body Large (Regular/Medium)**：农历、八字
- **Body (Regular)**：英文月周、警语、福语

### 字体原则
- **克制字号**：不要过大（主日期 < 0.25 height）
- **字重对比**：Display vs Body 层次分明
- **中文支持**：必须使用支持中文的字体

### 禁止事项
- ❌ Inter、Roboto、Open Sans（太通用）
- ❌ 只用 Regular 和 Bold（缺少 Medium）
- ❌ 过大字号（> 0.3 height）
- ❌ 不支持中文的字体（FreeSerif、FreeSans）

## 3. 间距系统

### 标准间距（DESIGN.md 标准）
```python
XS = 4    # 最小间距
SM = 8    # 小间距
MD = 16   # 中等间距
LG = 24   # 大间距（内边距）
XL = 32   # 超大间距（主要元素）
XL2 = 48  # 超超大间距（大区块）
```

### 使用规则
- **元素之间**：使用 XL (32px) 或 XL2 (48px)
- **内边距**：使用 LG (24px)
- **留白区域**：使用 XL2 (48px) 或更大

### 间距原则
- **翻倍规则**：当不确定时，将间距翻倍
- **留白充足**：留白 > 40% 画面
- **不要拥挤**：宁可少放，不要拥挤

## 4. 布局系统

### VL 驱动布局
**必须先分析背景图，再确定布局！**

#### 分析流程
1. 使用 `analyze_background.py` 分析背景图
2. 获取垂直/水平区域的暗色像素比例
3. 确定空白区域和图形区域
4. 根据分析结果调整文字位置

#### 布局原则
- **避开图形**：文字必须放在空白区域
- **不互相遮挡**：文字不能与背景图形重叠
- **不对称布局**：不要居中对齐，要偏左或偏右
- **大面积留白**：留白 > 40% 画面

#### 典型布局
- **左侧布局**：所有文字在左侧（8-12%），右侧留白
- **右侧布局**：所有文字在右侧（88-92%），左侧留白
- **顶部布局**：文字集中在顶部，底部留白
- **混合布局**：上下结合，但必须避让图形

### 禁止事项
- ❌ 居中对齐所有文字
- ❌ 不分析背景图就排版
- ❌ 文字和图形重叠
- ❌ 三个等宽卡片布局

## 5. 组件系统

### 组件定义

#### Date Card（日期卡片）
```
包含：
- 主日期（Display）
- 农历（Body Large）
- 八字（Body Large）
- 英文月周（Body）

位置：上部（10-40%）
```

#### Warning Blessing Block（警语福语区块）
```
包含：
- 警语（Body，6字）
- 福语（Body，6字）

位置：中部（55-70%）
格式：6+6 字对仗
```

### 组件原则
- **极简设计**：不要过度装饰
- **信息克制**：只保留最核心信息
- **层次分明**：Display > Body Large > Body
- **不对称放置**：不要居中对齐

## 6. 内容规范

### 必须包含
1. **主日期**：如 10
2. **农历**：如 二月廿三
3. **八字**：如 丙午·壬辰·甲寅
4. **警语**：6字，基于八字合盘
5. **福语**：6字，基于八字合盘
6. **英文月周**：如 APR FRI（可选）

### 警语福语生成
- **格式**：6+6 字对仗
- **来源**：基于八字合盘分析
- **内容**：
  - 警语：提醒注意事项
  - 福语：鼓励积极方向
- **示例**：
  - 警语：木土相克慎行
  - 福语：静守祈福安康

### 内容原则
- **对仗工整**：警语和福语字数相等、结构相同
- **基于合盘**：必须先做八字合盘分析
- **不重复**：不要在农历和八字中重复干支
- **克制表达**：不要冗长文字

### 禁止事项
- ❌ 直接列出宜忌列表
- ❌ 不基于合盘的警语福语
- ❌ AI 写作陈词滥调："Elevate", "Seamless"
- ❌ 冗长文字（警语福语 > 6字）

## 7. 工作流程

### 完整工作流

#### Step 1: 信息获取
- 获取今日干支
- 从黄历网站获取宜忌

#### Step 2: 八字合盘分析（重要！）
- 弗尼八字：乙丑 丙戌 戊申
- 今日八字：丙午 壬辰 甲寅
- 分析五行相生相克
- 输出合盘结论

#### Step 3: 改写警语福语
- 格式：6+6 字对仗
- 基于合盘结果生成

#### Step 4: 生成背景图（可选）
- 工具：coze-image-gen
- 提示词：**不包含可能生成文字的词汇**
- 尺寸：1080x1920

#### Step 5: VL 分析（必须！）
- 运行 `analyze_background.py`
- 获取图形分布
- 确定空白区域

#### Step 6: 智能排版（根据 VL 结果）
- 避开图形区域
- 不对称布局
- 大面积留白

#### Step 7: 合成海报
- 使用本设计系统的所有规范
- 色彩、字体、间距、布局

#### Step 8: 发送到美学群
- 只发送海报
- 不发送完整黄历

### 关键步骤
- ✅ **八字合盘分析**（不能跳过）
- ✅ **VL 分析背景图**（不能跳过）
- ✅ **根据 VL 结果智能排版**（不能跳过）

## 8. 质量检查清单

### 发送前检查

- [ ] 字体支持中文（Noto SansCJK）
- [ ] 色彩单一，饱和度 < 80%
- [ ] 字号克制（主日期 < 0.25 height）
- [ ] VL 分析已运行
- [ ] 文字避开图形区域
- [ ] 不对称布局
- [ ] 大面积留白（> 40%）
- [ ] 警语福语是 6+6 字对仗
- [ ] 警语福语基于八字合盘
- [ ] 没有紫/蓝 "AI 渐变"
- [ ] 没有纯黑色 `#000000`

### 常见错误

❌ **字体问题**
- 使用不支持中文的字体
- 只用 Regular 和 Bold
- 字号过大（> 0.3 height）

❌ **布局问题**
- 所有文字居中
- 不分析背景图就排版
- 文字和图形重叠
- 三个等宽卡片

❌ **内容问题**
- 警语福语不是 6+6 字对仗
- 不基于八字合盘
- 直接列出宜忌列表

❌ **视觉问题**
- 过饱和颜色
- 紫色/蓝色渐变
- 纯黑色背景
- 缺少留白
